# RP CLI

RP CLI is a Zod-based command line state runtime framework for AI agents.

The current repository contains the design documents and the initial TypeScript workspace scaffold. See:

- [docs/NEED.md](docs/NEED.md)
- [docs/PLAN.md](docs/PLAN.md)
- [docs/PROGRESS.md](docs/PROGRESS.md)

## Development

```bash
npm install
npm run typecheck
npm test
```

The first implementation target is the core runtime foundation described in `docs/PROGRESS.md`.
